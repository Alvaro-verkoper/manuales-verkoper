1. Objetivo

Definir términos técnicos y operativos para interpretación uniforme.

2. Comerciales

Cliente Frecuente: Cliente con más de 5 compras en los últimos 12 meses.

Cotización: Propuesta comercial no vinculante que detalla productos, precios y condiciones.

Nota de Venta (NV): Documento interno que registra compromiso de venta con todos los detalles comerciales y logísticos.

Venta Efectiva: Operación cerrada con pago confirmado y documentación completa.

3. Administrativos/Documentales

Guía de Despacho (GD): Documento tributario que respalda el traslado de mercadería.

Factura: Documento tributario que formaliza la venta y permite el cobro.

Orden de Compra (OC): Documento formal de solicitud de productos a proveedores.

Flujo Documental: Secuencia obligatoria de documentos: Cotización → NV → GD → Factura.

Trazabilidad: Capacidad de rastrear y respaldar cada paso de un proceso mediante documentación y registros.

4. Logística/Bodega

Despacho: Envío de mercadería desde bodega a destino mediante transporte contratado.

Retiro: Gestión donde personal de Verkoper recoge mercadería desde proveedor.

Producto Sobredimensionado: Mercadería que excede dimensiones de pallet estándar (1.0 x 1.2 metros).

Registro Fotográfico: Evidencia visual obligatoria del estado de mercadería en movimientos críticos.

5. Seguridad

Correo Sospechoso: Mensaje electrónico con señales de fraude: dominio alterado, urgencia extrema, enlaces desconocidos.

Estafa: Intento de engaño para obtener beneficios económicos o información sensible.

Verificación Telefónica: Llamada obligatoria para confirmar identidad y legitimidad de cliente/solicitud.

Escalamiento: Proceso de informar situación anómala a nivel superior de autoridad.

6. Laborales/Disciplinarios

Amonestación Verbal: Llamado de atención oral por incumplimiento leve.

Amonestación Escrita: Carta formal que registra incumplimiento grave.

Falta Leve: Incumplimiento menor sin impacto operativo significativo.

Falta Grave: Incumplimiento reiterado o con impacto operativo moderado.

Falta Gravísima: Incumplimiento con impacto económico, riesgo significativo o reincidencia extrema.

Cargo Crítico: Función cuyo incumplimiento afecta directamente la operación (ej: apertura bodega).

7. Sistemas/Herramientas

ERP MAXXA: Sistema de gestión empresarial utilizado para ventas, facturación, stock y reportes.

Marcaje Biométrico: Sistema de registro de asistencia mediante huella digital.

Documento de Seguimiento: Excel o archivo digital para control de gestiones pendientes y estados.

8. Vigencia

Complementa todos los documentos oficiales del sistema de gestión documental.
